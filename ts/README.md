# ts

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run start
```

# CLI Usage

1. visit circles & get your PRIVATE_KEY from local storage (or any owner key).
2. cd ts
3. cp sample.env .env (and fill out your keys) `SAFE_ADDRESS`, `PRIVATE_KEY`
4. source .env
5. bun run start
